# Loyalty 积分规则配置平台设计文档
> **版本**：1.0\
> **关联主文档**：基于AI的下一代全渠道忠诚度管理SaaS平台 v7.3\
> **设计目标**：为运营人员提供一站式的积分规则配置、测试、发布平台，涵盖规则编写、模拟测试、集成验证、提示词管理全流程。以“降低配置门槛、提升交付质量”为核心目标，所有操作围绕“自然语言描述 → AI解析 → 模拟测试 → 集成验证 → 发布上线”的主线展开。

## 七、规则测试与提示词管理数据模型
### 7.1 规则测试套件表
sql
```
CREATE TABLE rule_test_suite (
    id VARCHAR(64) PRIMARY KEY,
    program_code VARCHAR(32) NOT NULL,
    suite_name VARCHAR(128) NOT NULL,
    description TEXT,
    rule_ids JSONB,                          -- 包含的规则ID列表
    test_data JSONB,                         -- 测试输入数据
    expected_results JSONB,                  -- 预期结果
    status VARCHAR(20) DEFAULT 'DRAFT',      -- DRAFT / READY / RUNNING / PASSED / FAILED
    created_by VARCHAR(64),
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);
```
### 7.2 规则测试执行记录表
sql
```
CREATE TABLE rule_test_execution (
    id VARCHAR(64) PRIMARY KEY,
    program_code VARCHAR(32) NOT NULL,
    suite_id VARCHAR(64),
    rule_id VARCHAR(64),
    test_type VARCHAR(20),                   -- SIMULATION / INTEGRATION / REGRESSION
    test_data JSONB,
    result JSONB,                            -- 执行结果（匹配状态、动作、计算过程）
    status VARCHAR(20),                      -- SUCCESS / FAILED / ERROR
    execution_time_ms INT,
    executed_by VARCHAR(64),
    executed_at TIMESTAMPTZ DEFAULT NOW()
);
```

sql
```
CREATE TABLE ai_prompt_config (
    id VARCHAR(64) PRIMARY KEY,
```

## 八、总结
| 功能模块           | 核心能力                     | 用户价值              |
| -------------- | ------------------------ | ----------------- |
| **规则配置（AI模式）** | 自然语言描述 → AI解析 → 结构化配置    | 降低配置门槛，运营无需学习技术   |
| **规则配置（表单模式）** | 可视化表单配置复杂逻辑              | 精细控制，支持阶梯、循环等高级场景 |
| **模拟测试**       | 单条规则验证，展示匹配过程和计算细节       | 快速验证规则正确性         |
| **集成测试**       | 多条规则协同验证，检测冲突            | 确保规则间无冲突，行为符合预期   |
| **提示词管理**      | System Prompt 配置、测试、版本管理 | AI生成质量可调优，支持多租户定制 |
| **变量管理**       | 表达式定义派生指标，规则中直接引用        | 屏蔽底层复杂性，提高规则可读性   |
| **规则生命周期**     | 草稿→测试→审核→发布→停用           | 保障规则质量，降低生产风险     |
| **版本管理**       | 规则版本记录，支持回滚              | 快速恢复，降低变更风险       |
